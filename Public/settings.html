In Overkill Mode, the remaining bits after a final blow determines the HP value of the next boss. As an example, if Kappa defeats PogChamp (who has 100 HP) with 250 bits, then Kappa will have 150 health. If the multiplier is set to 2, then Kappa will start with 150 x 2 = 300 HP.

In Progressive Mode, the first Bit Boss starts with an initial amount of HP, and each successive boss afterward has an incrementally higher HP value.

In Constant Mode, each Bit Boss will always have the same amount of starting HP.

NOTE: Changing HP modes resets the current boss and their HP.
